# End-to-End Banking Analytics
End-to-end Banking Analytics Dashboard built with Python and Power BI, featuring 2M+ synthetic records, star schema modeling, and interactive business insights.

An enterprise-grade Banking Business Intelligence Dashboard built in Power BI using a Star Schema Data Model, DAX, and Interactive Visualizations.

This project demonstrates real-world banking analytics using realistic synthetic data.

## Project Overview

This project demonstrates how a commercial bank can monitor its overall business performance using a centralized Business Intelligence Dashboard.

The dashboard enables executives to monitor:

- Deposits
- Loan Portfolio
- Customer Analytics
- Branch Performance
- Product Performance
- Risk & NPL
- Profitability
- Executive KPIs
- Predictive Analytics

The dataset used in this project is synthetic (dummy) but designed to closely resemble real-world banking operations.

## Data Model

The project follows a Star Schema.

Dimensions

- Dim Date
- Dim Customer
- Dim Branch
- Dim Product
- Dim Region
- Dim Employee

Fact Tables

- Fact Deposit
- Fact Loan
- Fact Transaction
- Fact Customer Activity
- Fact Profitability
- Fact NPL
- Fact Product Performance
- Fact Branch Performance
- Fact Risk Score

## Synthetic Data

The datasets were generated using Python.

The data simulates realistic banking operations including

- Deposits
- Loans
- Transactions
- Profitability
- Risk
- Branches
- Customers

No confidential or proprietary banking data has been used.
